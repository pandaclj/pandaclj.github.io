小芒果の出生日记

背景篇

 我，一个普通的不能再普通的杭漂程序猿。我老婆，一个事业单位的工作人员。我，大部分时间在杭州；我老婆，一直在绍兴。但我们的生活重心都在绍兴，因为家在绍兴。

检查篇

 从得知老婆怀孕开始，我们的生活就开始围绕着小芒果了，虽然此时的她还在肚子里，但总幻想着她出来的样子。整个怀孕过程中，大致会经历几个比较重要的检查期，NT -> 唐筛 -> 三维大排畸 -> 四维 -> 糖耐 ->助产士门诊。每一次检查前，心总是被吊着，检查过后就像考完试一样松一口气，当然这个只是暂时的，因为整个孕期都得小心翼翼，这将是一场持久战役。
 第一次和小芒果见面，是去医院检查做B超确定怀孕的那天。那天，小芒果还只是豆子般大小，在B超报告单上，我甚至第一时间都没发现她，要不是下面写着一排字，我都不知道我要当爸爸了。看着报告单，我们内心都很激动，因为这意味着一个新的生命即将走进我们的生活，她将是我们未来的生活中心。第二次比较正式的见面，就是做NT那天了，开心的是NT一次过（可能这也是小芒果最给力的一次了=.=，因为后面无论做什么检查，四维也好，胎心也好，她总是不配合）。遗憾的是，对于小孩的轮廓很模糊，不像其他人做的检查，很清晰，所以对她的样子，我还一直期待着。第三次见面，是做三维的时候，三维是静态的，做出来的照片显示也没有理想中的清晰，于是只能把宝都压在了四维上。因为妇保院没有四维机器，我们去的私立医院做四维，这一次，我第一次看清了小芒果长什么样，胖嘟嘟的脸，嘴巴一直吮吸着自己的手指，要么就是在那睡觉，极度不配合，这也导致四维我们是最早进去，确是最后一个做完的=.=，中间不通过后得通过不停地吃东西和爬楼梯来让小芒果活跃起来，还买了棒冰给她吃，看来在肚子里就得好好供着。好在结果满意，小芒果一切都很好，很期待她的出生。
 四维过后，差不多6个多月了。之后就要开始进行频繁的产检了。周期越久，产检频次会越高。而且每次都得花蛮长时间，为了让老婆有一次好的检查分娩体验，加上我们这的妇保院也第一次推出了“产科一体化”，在商量之后，我们也是在第一时间报了这个。从之后的经历来看，这个还是值得的。产检无非几个点，观察妈妈的一些生理状况(血压)，观察宝宝的羊水，胎心，身体大小，之后就是静静地期待小芒果的降临。

备货篇

从四维过后，我们就进入了紧张的备货阶段。为此，我们也是查阅了大量的资料。这边我就筛选出一些我们认为有用当然之后也确实用到的（后面也有一些是太突然或者没考虑到而到医院附近的母婴店买的，价格实在是太暴力了），所以还是早点准备，准备的周全些比较好。

妈妈物品

时期 物品
分娩时 NB尿不湿一片、中单（产褥垫）、毛巾、纸巾、产妇专用卫生纸、充电宝（多备几个)，给妈妈的补充能量的食物和饮料，婴儿帽
分娩后 双熊猫卫生纸（硬）、软的卫生纸也要准备，中单，孕妇帽

宝宝物品

恒温调奶器、奶瓶沥干盒、奶瓶（多备几个）、吸奶器、奶粉、尿不湿（NB备个3，4袋）、妈妈包、宝宝湿巾、宝宝柔纸巾、外出奶粉携带盒、储奶袋。。。之后宝宝的东西可能会越卖越多

取名篇

我们这取名字一来不能跟长辈或者同辈有重字，二来还要根据出生的时辰算出来五行缺啥，缺啥名字上得带上那个字。所以，我们是想了好多备选名字，而且男孩女孩名字都得想。期间还互相嫌弃对方取的名字不好听（^.^）。这个也是尽快想好，等出生后就可以马上确定下来，方便快点去办理一些相关证件。如果想让宝宝的名字特别些，也可以去浙里办查下重名，看看省内有多少一样的。

分娩篇

 由于两地的关系，我基本是每周回家一次。所以当预产期越来越近的时候，我的内心总是很忐忑。我曾经常梦到老婆突然要生，然后我用各种方法紧急赶回家的样子。本来也预想着陪产假能刚好和国庆长假衔接，因为预产期是9月中旬，这样也能有更多的时间照顾老婆做月子。老婆还想着在宝宝出生前能出去吃一顿烤肉，一顿火锅。可惜，往往理想很美好，现实却啪啪打脸。才37w+1，她就想出来了。8.30那晚刚好周五我从杭州回家。然后31日凌晨的时候，老婆突然把我叫醒，说羊水膜破了。虽然，早给了自己一些遇到这些事情怎么应对的策略。可是真的遇到了，我还是很惊慌，脑袋一片空白，不知道该怎么办。因为我知道，羊水破比见红更紧急，羊水破意味着必须在一定时间内让宝宝出生，不然后果很严重。好在老婆比较镇定，打了120叫救护车（老婆同事接的电话），我紧急准备一些待产的物品，去医院需要用到的东西，证件等（本来这些打算周末整理的，计划不如变化，早点准备）。等待救护车到来的过程真的很煎熬（然后发现来的医生护士还是我老婆认识的，瞬间安心），话说这还是我第一次坐救护车，中间还应为忘记带证件了，让救护车又中途回去了一次，尴尬。
 到了医院急诊，我就负责办理一些入院手续，然后去病房，一直进行胎心监听。期间，羊水不停地流出来，我的心一直悬着。就这样，整夜没睡，第二天一大早又匆匆赶回家取一些忘带的物品（凌晨事发突然，太慌乱了）。到了早晨医生上班后，老婆就准备进入分娩室开始分娩了。而我们，就只能静静地在分娩室外的座位上等着，唯一能知道里面发生什么的只有门口的一块显示屏，上面显示着xxx开了多少指，进行到了第x产程。从早上，一直等到晚上5点多，期间只能通过不断地来回踱步来缓解自己紧张的心情。看着一个一个婴儿从里面出来，而老婆的分娩状态仍然是待产中（刚进分娩室的状态就是待产中），心里越发的着急，同时整个人也是很疲倦，眼皮不停地打架，可是没法睡着。中间的过程就不过多描述了，经历过的都懂。终于，到了18点左右，说开了两指打无痛了，然后家属可以进去了，心里终于安心了点。进去后过了很久，宫缩一直不明显，开指也一直停留在原来的阶段，因为过去的时间太久了，又怕羊水流完，也怕顺转剖，所以很担心。之后只能通过不断的增加催产素的浓度，减少甚至关掉无痛，开指才慢慢的增加。一直经历了几小时的持续疼痛，差不多全开了，进入了第二产程。助产士不停的鼓励，指导，老婆也很争气，小芒果也是一点点的出来。中间过程就省略了，总之一个结论，母亲很伟大，好好善待自己的母亲，好好疼爱自己的妻子。这个过程，毕生难忘。好在宝宝平安顺利的出生，差一个小时就9月1号了，我知道，她是想上学了（^.^）。

监护室篇

可能是产程太长，宝宝吸了不少羊水，出来时呻吟声让人听着很揪心。护士也是给她吸了两管羊水出来，可是效果并不明显。所以，在见了一面后宝宝就被送进了新生儿监护室。凌晨，医生抱着宝宝，我跟在后面，办完了入院手续后，去监护室门口签了一大堆纸，四周寂静，听了医生说的对宝宝的一些可能的病情，眼眶止不住湿润。签完字，独自回到了分娩室，这段路感觉格外的长。在监护室，是不允许家属探视的，能做的就是在一、三、五允许问下病情。对于家长来说，这几天过的格外的漫长，一切的未知，总让人的心悬着。对于宝妈，也是格外的难受，好几次到了晚上就不停的抹泪。当然，对于大部分宝宝，都还是比较顺利的，可能就没有这一篇章了。

证件篇

证件 所需材料 备注
准生证 双方身份证信息 打算怀孕的时候，就可以去办理准生证了，这个可以在办理
出生证明 双方身份证原件，落户户口本，小孩出生信息的一张纸
户口 双方身份证原件，户口本，出生证明
接种证 双方身份证原件，户口本，出生证明，医院打疫苗证明 乙肝第一针和卡介疫苗，卡介疫苗如果小孩指标不达标，不给打，比如黄疸高，这个后续也可以去地方卫生院办理
医保 双方身份证原件，户口本，出生证明 这个很重要，如果宝宝出生后需要住院，出院后可报销，尽快办理

爸爸篇

照顾宝妈，照顾宝宝。
照顾宝妈最多的四件事：洗碗、泡红糖水、吹头发、送餐到房间
照顾宝宝最多的四件事：洗奶瓶、换尿布、冲奶粉、拍嗝

这期间的宝爸就是要像超人一样，只要有事发生，立马能出现解决。

祝愿篇

此文只是我对自己这段时间经历的一个描述，同样也献给未来的新爸新妈们，希望看了此文对你们有所帮助。最后，祝愿大家都能有个健康的宝宝，祝愿我们的宝宝都能健康、快乐地成长。也希望我们能多多交流
